STANDARD UNIVERSITY COLOR PALETTE

The palette contains the core colors that should be used to match with the University combination logo across all University’s communication channels.

The following regulations apply to the usage of HKUST brand.

University Combination Logo Colors Accent Colors Colors to be Avoided
+

UNIVERSITY COMBINATION LOGO COLORS

When using the monochrome logo, choose from one of the following six colors only. Do not use any other colors.

DESKTOP PUBLISHING

For offset printing or silk-screen, it is suggested using either PANTONE or PROCESS COLORS (CMYK).

ELECTRONIC PRESENTATION

For PowerPoint presentation or digital prints, it is suggested using RGB colors.

ONLINE PUBLISHING

For web pages, it is suggested using web colors.

FABRIC PRINTING

For fabric printing, it is suggested using PANTONE or PANTONE for fabric (if applicable)

Pantone – Matching system (PMS) A numbered color system that printers and designers use to indicate color choice. For more information, please visit Pantone website.

Process colors – Cyan, Magenta, Yellow and Black inks (abbreviated as CMYK, with black symbolized as ”K”) used in the four-color offset printing process.

RGB – (Red, Green and Blue) The colors used in computer monitors.

Web safe colors – 216 colors that appear consistently, without breaking up and dithering, across web browsers and across computer platforms.

ACCENT COLORS

These accent colors are designed to work harmoniously with the University combination logo colors. They can be used in bars, panels, tabs, headings, subtitles, and graphs. Users are welcome to use any other color that is most desirable for any occasion except for the theme colors that are regularly used by other local universities. Users are recommended to avoid using these color combinations which are listed on the next section.

COLORS TO BE AVOIDED

It is recommendable to avoid using the following color tones which are currently used by other local universities as their theme colors.

設計與應用

大學標準用色

色板包含用來配襯大學複合式標誌的主色調，適用於香港科技大學品牌的所有通訊。

以下規定適用於香港科技大學的品牌形象。

大學複合式標誌用色 強調顏色 避免使用的顏色
+

大學複合式標誌用色

當使用單色標誌時，只可從下列六款顏色中選擇，不應自行挑選其他顏色。

桌面排版

凡平版印刷或絲網印刷，建議使用「彩通」或「印刷四分色模式」（CMYK）。

電子發佈

凡以PowerPoint 展示或數位印刷，建議採用「三原色光模式」(RGB)。

網絡出版

於網頁上建議使用網頁通用顏色。

布料印刷

凡布料印刷，建議使用「彩通」(PANTONE) 或「彩通-布料」(PANTONE for Fabric) (如適用)。

彩通配色系統（PMS）：以色彩編號系統，供印刷商和設計師選擇顏色。如需更多資訊，請瀏覽彩通網站。

印刷四分色模式：以青色、洋紅色、黃色和黑色墨水（英文縮寫為 CMYK，K 代表黑色）進行四色平版印刷。

三原色光模式：分別是紅、綠、藍色，亦即電腦螢幕所採用的三原色。

網頁安全顏色：在所有網頁瀏覽器和電腦平台上，連貫一致地出現且不會分散和抖色的 216 種顏色。

大学复合式标志用色

桌面排版

凡平版印刷或丝网印刷，建议使用"彩通"或"印刷四分色模式"（CMYK）。

(PANTONE for Fabric) (如适用)。

印刷四分色模式：以青色、洋红色、黄色和黑色墨水(英文缩写为 CMYK，K 代表黑色）进行四色平版印刷。

三原色光模式：分别是红、绿、蓝色，亦即计算机屏幕所采用的三原色。

网页安全颜色：在所有网页浏览器和计算机平台上，连贯一致地出现且不会分散和抖色的 216 种颜色。
