ACCESSIBILITY

BROWSERS

The website has been tested on most of today’s leading browsers, for optimum viewing of the HKUST site we recommend a screen resolution of 1920×1080.

The HKUST website has been tested in the following browsers:

  • Chrome 33.0 and above
  • Firefox 27.0 and above
  • Internet Explorer 11 and above
  • Safari (Macintosh) 5.0 and above
  • iOS 6.0 and above (Mobile Safari)
  • Android Browser 4.0 and above

FONT SIZE

The HKUST website offers three level of font size (AAA). Users can rescale the font size of this website with the font size control tool on the top of the webpage. If the largest font size does not suit your needs, you can enlarge with the “Zoom In” function of the browser.

PLUGINS

We also use PDFs in places as a standard downloadable document. If you require the latest version of Adobe Acrobat Reader, you can get a copy from the Adobe Website.
